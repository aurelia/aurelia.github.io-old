Aurelia Bug Fixes, More Docs and New Skeletons!

Posted by AureliaEffect on January 8, 2016

Today's patch updates come with the standard set of bug fixes. However, this week we've added five new articles to the docs and two new skeletons for Visual Studio users.

Bug Fixes

This release contains numerous bug fixes, stability improvements and edge-case handling. To get the fixes you can run jspm update. You'll be able to continue to run jspm update to get our latest fixes and improvements until we release Beta 2. You also should have no fear of breaking changes. If you encounter anything, then it's an accidental regression and something we would love for you to help us correct by creating an issue with the regression's details.

You can find the release notes related to today's patches at the end of this post.

Docs

We've added five new articles to the docs:

  • Binding: Value Converters
  • Binding: Binding Behaviors
  • End-to-End Testing
  • Securing Your App
  • Integrating with Polymer (under the new integrators profile)

In addition to the articles, we've also added a retractable right-side panel that will display the table of contents for each article. This allows you to see the article's contents at a glance, skip around and deep link to sections within an article. We'll continue to add more articles and enhance the capabilities of our docs regularly.

Skeletons

From the beginning, we've had an ES 2016 skeleton. Recently we added a TypeScript skeleton. With this set of releases, we now have ES 2016 and TypeScript skeletons specifically configured for Visual Studio and ASP.NET 5. As part of this release, we've moved all skeletons to the Skeleton Navigation repository for convenience. Simply download the latest release and select the skeleton that suits you best.
