Aurelia on .NET Rocks

Posted by AureliaEffect on April 6, 2016

If you are a .NET developer...or really any developer interested in hearing some discussion on Aurelia, check out the latest .NET Rocks episode .

What will you hear in this episode? Well, we got to talking about the importance of Convention over Configuration and how Aurelia's approach allows you to write much of your code as Vanilla JS, without unnecessary framework intrusion. We talked about features like our View Engine Pipeline, how you can use CSS Modules, React and even Angular components all together in Aurelia. We even had a short discussion on what's happening with the Extensible Web Manifesto, the CSS Houdini Working Group and future plans for Aurelia such as server-side rendering and Aurelia Interface.

As usual, it was a blast talking with Carl and Richard . I hope you all enjoy it as much as I did.
