Aurelia Release Notes - Early July 2017

Posted by AureliaEffect on July 7, 2017

The Aurelia Team continues to iterate and improve all aspects of the framework. Below you can find details on our recent minor release of aurelia-validation along with our patch releases of the aurelia-cli and aurelia-polyfills.

Minor Releases

Minor updates include new features, updates to bugs and performance improvements.

aurelia-validation 1.1.0

Bug Fixes

  • getPropertyInfo: use scope traversal ( 6789112 ), closes #408
  • ValidationControllerFactory: supply required arg ( 377e6c9 )
  • ValidationRules: merge rules for same property ( 33f741a ), closes #400
  • ValidationRules: satisfies callback arg required ( 258615e ), closes #426

Features

  • build: add ES2017 target ( b266af5 )
  • ValidationController: accept property accessor expression in addError ( 842fb4c ), closes #422
  • ValidationController: add revalidateErrors method ( 47026ec ), closes #349
  • ValidationController: events ( 57232ce ), closes #318
  • ValidationRules: untaggedRules ( 549a4a9 ), closes #436

Patch Releases

Patch updates include only fixes to bugs and performance improvements.
