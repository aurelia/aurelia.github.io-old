Aurelia Release Notes - Early February 2018

Posted by Rob Eisenberg on February 5, 2018

Continuing into 2018, we've got a number of fixes and improvements to core Aurelia libraries and plugins. The details are available below.

Minor Releases

Minor updates include new features, updates to bugs and performance improvements.

aurelia-router 1.5.0

Bug Fixes

  • navigation-instruction: Wait to swap child routes ( 1432234 )

Features

  • router: Enhance optional view port feature ( bcf4e33 )
  • router: optional viewports ( 8878b45 )
  • router: viewport defaults ( 0bfc4bc ), closes aurelia/router#482
  • router config: ignoreUnknownRoutes option ( 811959b )

aurelia-templating-router 1.3.0

Features

  • templating-router: optional viewports ( 24532ff )

aurelia-fetch-client 1.3.1

Bug Fixes

  • http-client: Rework application/json header ( 946273a ), closes #90
  • util: Discontinue using Blob for JSON ( 03ae35f ), closes #90

aurelia-polyfills 1.3.0

Features

  • Object: add Object.is() ( #58 ) ( 3cf3410 )

aurelia-pal 1.5.0

Features

  • dom: Enable injectStyles to replace the contents of an existing style element by id.

aurelia-pal-browser 1.4.0

Features

  • dom: Enable injectStyles to replace the contents of an existing style element by id.

Patch Releases

Patch updates include only fixes to bugs and performance improvements.

aurelia-i18n 2.1.1

Bug Fixes
