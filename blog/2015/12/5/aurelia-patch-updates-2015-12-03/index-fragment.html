Aurelia Patch Updates (2015-12-03)

Posted by AureliaEffect on December 3, 2015

Now that the Beta is out, we're focusing our time on bug fixes, performance and polish. This week we've released a number of patches to libraries that fix bugs and enhance performance in certain scenarios. To get the latest, simply run jspm update. You can find the release notes below. Stay tuned for tomorrow's post on performance...
