Get Familiar

Use our guides to jumpstart your Aurelia development.

Our quick start is a great place to get your feet wet with the Aurelia framework. It's created to allow you to explore and tinker with very little setup time. After that, check out all the awesome topical guides. For a more interactive experience, join the Aurelia gitter channel. Feel free to join the conversation and ask our contributors and friendly, knowledgeable community questions as you begin developing with Aurelia.

Keep an eye on the offical Aurelia blog. New features, Aurelia news and industry announcements will be posted there to keep the community up to date.

Check out the API Docs!

Straight from source.

Built straight from source, Aurelia’s API docs will give you details on how to use each of the modules. This documentation stays updated with release builds, and is always up to date with the latest features of Aurelia.
