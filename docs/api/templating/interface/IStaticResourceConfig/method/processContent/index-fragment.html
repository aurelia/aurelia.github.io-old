processContent

A Method

processContent(viewCompiler: ViewCompiler, resources: ViewResources, node: Element, instruction: BehaviorInstruction): boolean

Enables custom processing of the content that is places inside the custom element by its consumer. Pass a boolean to direct the template compiler to not process the content placed inside this element. Alternatively, pass a function which can provide custom processing of the content. This function should then return a boolean indicating whether the compiler should also process the content.

Parameters

viewCompiler: ViewCompiler
resources: ViewResources
node: Element
instruction: BehaviorInstruction
