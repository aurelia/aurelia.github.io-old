create

A Method

create(container: Container, instruction?: BehaviorInstruction, element?: Element, bindings?: any): Controller

Creates an instance of this behavior.

Parameters

container: Container
The DI container to create the instance in.
instruction?: BehaviorInstruction
The instruction for this behavior that was constructed during compilation.
element?: Element
The element on which this behavior exists.
bindings?: any
The bindings that are associated with the view in which this behavior exists.

Returns

The Controller of this behavior.
