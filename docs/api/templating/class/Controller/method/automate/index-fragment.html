automate

A Method

automate(overrideContext?: Object, owningView?: View): void

Used to automate the proper binding of this controller and its view. Used by the composition engine for dynamic component creation. This should be considered a semi-private API and is subject to change without notice, even across minor or patch releases.

Parameters

overrideContext?: Object
An override context for binding.
owningView?: View
The view inside which this controller resides.
