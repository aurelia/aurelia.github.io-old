removeMany

A Method

removeMany(viewsToRemove: any, returnToCache?: boolean, skipAnimation?: boolean):

Removes many views from the slot.

Parameters

viewsToRemove: any
The array of views to remove.
returnToCache?: boolean
Should the views be returned to the view cache?
skipAnimation?: boolean
Should the removal animation be skipped?

Returns

May return a promise if the view removal triggered an animation.
