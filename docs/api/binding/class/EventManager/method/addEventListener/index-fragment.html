addEventListener

A Method

addEventListener(target: EventTarget, targetEvent: string, callback: EventListenerOrEventListenerObject, delegate: delegationStrategy, disposable: true): Disposable

Subscribes to specified event on the target element.

Parameters

target: EventTarget
Target element.
targetEvent: string
Name of event to subscribe.
callback: EventListenerOrEventListenerObject
Event listener callback.
delegate: delegationStrategy
True to use event delegation mechanism.
disposable: true
True to return a disposable object with dispose() method instead of a function

Returns

function which removes event listener.
