Books

There are a number of books available to help you get started learning Modern JavaScript and Aurelia:

  • Aurelia For Real World Web Applications by Dwayne Charrington
  • Aurelia in Action by Sean Hunter
  • Learning Aurelia by Manuel Guilbault
  • Practical App Development with Aurelia by Matthew Duffield
  • Aurelia Succinctly by Matthew Duffield
  • Practical Aurelia by Behzad Abbasi

Videos
